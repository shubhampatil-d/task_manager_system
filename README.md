## How to Run the Project

### 1. Clone the repo
git clone https://github.com/shubhampatil-d/task_manager_system.git
cd task_manager


### 2. Start Backend
cd backend
npm install
npx ts-node src/index.ts

Backend will run on:
http://localhost:5000


### 3. Start Frontend (open new terminal)
cd frontend
npm install
npm run dev

Frontend will run on:
http://localhost:3000


### 4. Open in browser
http://localhost:3000/register

- Register a user  
- Login  
- Use dashboard  

---

Note:
Make sure both backend and frontend are running at the same time.
